# iBroadcast App

Forked from <https://github.com/Nic128/iBroadcast-desktop>.

Electron wrapper for [iBroadcast web player](https://media.ibroadcast.com).

## Support

- Mac ✅
- Windows ✅
- x64 & ARM64 ✅

## Development

```
yarn && yarn start
```

Build the app for your system:

```
yarn app:dist
```

Binaries should be in the dist folder.
